# Lot Split Feasibility Engine — Phase 2: Parcel Data Adapter

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a fully generic, database-driven ArcGIS REST parcel adapter. All jurisdiction-specific configuration (GIS URL, field mappings, zoning code map) lives in the `Jurisdiction` DB record — not in Python files. Adding a new jurisdiction in a future phase means inserting a DB row and running a seed script; it requires zero new Python code.

**Architecture:**
- `Jurisdiction` model gets three new JSONB/Text columns: `gis_feature_server_url`, `gis_field_map`, `gis_zoning_code_map`
- `JurisdictionConfig.from_orm(jurisdiction)` factory converts a DB row into the in-memory config object
- `ArcGISParcelAdapter(config)` is a single, fully generic class — no subclassing per county
- Kyle/Hays County enters the system via `scripts/seed_hays_county.py` (data, not code)

**New external dependencies:** `requests>=2.31`, `python-dotenv>=1.0`
**New dev dependencies:** `pytest-mock>=3.12`
**Infra required:** PostgreSQL 16 + PostGIS 3.4 (Docker Compose provided in Task 0)

---

## Global Constraints

- `app/adapters/` may import from `app/models/` but NOT from `app/engine/` — adapters are the DB-facing layer; engine remains isolated
- No hardcoded field names or URLs in `app/adapters/` — all per-jurisdiction data comes from the `Jurisdiction` DB record
- Area must be computed from geometry via `pyproj.Geod`, NOT trusted from county-reported fields (per spec §5.1)
- All HTTP calls live inside `ArcGISParcelAdapter` — no `requests` calls anywhere else
- If county GIS returns no feature for an APN, the adapter returns `None` — never raises
- `Parcel.zoning_district_id` may legitimately be `NULL` if the raw code has no mapping entry — explicit data gap, not an error
- Database session management is the caller's responsibility — the ingestion service accepts a `Session`, never creates one

## Task 3 — Adapter base types (`app/adapters/base.py`)

```python
@dataclass
class ParcelRecord:
    """Normalized fields from a county data source, before DB upsert."""
    apn: str
    geometry_geojson: dict          # GeoJSON Polygon, EPSG:4326
    address_normalized: str | None
    zoning_code_raw: str | None
    owner_name: str | None
    assessed_land_value: float | None
    assessed_improvement_value: float | None
    last_sale_price: float | None
    last_sale_date: date | None
    existing_structures_count: int
    raw_source_data: dict           # full raw payload, stored in Parcel.raw_assessor_data

@dataclass
class FieldMapping:
    """Canonical field role → county GIS attribute name. None = field absent in this layer."""
    apn: str
    address: str | None = None
    zoning_code: str | None = None
    owner_name: str | None = None
    assessed_land: str | None = None
    assessed_improvement: str | None = None
    last_sale_price: str | None = None
    last_sale_date: str | None = None
    structures_count: str | None = None

    @classmethod
    def from_dict(cls, d: dict) -> "FieldMapping":
        return cls(
            apn=d["apn"],
            address=d.get("address"),
            zoning_code=d.get("zoning_code"),
            owner_name=d.get("owner_name"),
            assessed_land=d.get("assessed_land"),
            assessed_improvement=d.get("assessed_improvement"),
            last_sale_price=d.get("last_sale_price"),
            last_sale_date=d.get("last_sale_date"),
            structures_count=d.get("structures_count"),
        )

@dataclass
class JurisdictionConfig:
    """Everything the generic adapter needs to query one jurisdiction's ArcGIS service."""
    jurisdiction_name: str
    feature_server_url: str
    field_mapping: FieldMapping
    date_format: str = "%Y-%m-%d"

    @classmethod
    def from_orm(cls, jurisdiction: "Jurisdiction") -> "JurisdictionConfig":
        """Build config from a Jurisdiction DB record. Raises ValueError if GIS not configured."""
        if not jurisdiction.gis_feature_server_url:
            raise ValueError(
                f"Jurisdiction '{jurisdiction.name}' has no gis_feature_server_url configured. "
                "Run the seed script for this jurisdiction before using the parcel adapter."
            )
        field_map = jurisdiction.gis_field_map or {}
        if "apn" not in field_map:
            raise ValueError(
                f"Jurisdiction '{jurisdiction.name}' gis_field_map is missing required key 'apn'."
            )
        return cls(
            jurisdiction_name=jurisdiction.name,
            feature_server_url=jurisdiction.gis_feature_server_url,
            field_mapping=FieldMapping.from_dict(field_map),
        )

class ParcelAdapter(Protocol):
    def fetch_by_apn(self, apn: str) -> ParcelRecord | None: ...
    def fetch_by_location(self, lat: float, lon: float) -> ParcelRecord | None: ...
```

## Task 4 — Generic ArcGIS REST adapter (`app/adapters/arcgis.py`)

```python
import logging
import requests

logger = logging.getLogger(__name__)

class ArcGISParcelAdapter:
    def __init__(self, config: JurisdictionConfig, timeout: int = 15):
        self._config = config
        self._timeout = timeout
        self._session = requests.Session()

    def fetch_by_apn(self, apn: str) -> ParcelRecord | None:
        apn_field = self._config.field_mapping.apn
        params = {
            "where": f"{apn_field}='{apn}'",
            "outFields": "*",
            "returnGeometry": "true",
            "f": "geojson",
        }
        return self._query(params)

    def fetch_by_location(self, lat: float, lon: float) -> ParcelRecord | None:
        params = {
            "geometry": f"{lon},{lat}",
            "geometryType": "esriGeometryPoint",
            "spatialRel": "esriSpatialRelIntersects",
            "inSR": "4326",
            "outFields": "*",
            "returnGeometry": "true",
            "f": "geojson",
        }
        return self._query(params)

    def _query(self, params: dict) -> ParcelRecord | None:
        url = f"{self._config.feature_server_url}/query"
        logger.debug("ArcGIS query: %s params=%s", url, params)
        response = self._session.get(url, params=params, timeout=self._timeout)
        response.raise_for_status()   # RuntimeError on 4xx/5xx
        fc = response.json()
        features = fc.get("features", [])
        if not features:
            return None
        if len(features) > 1:
            logger.warning(
                "ArcGIS returned %d features for query; using first. params=%s",
                len(features), params,
            )
        return self._parse_feature(features[0])

    def _parse_feature(self, feature: dict) -> ParcelRecord:
        props = feature.get("properties") or feature.get("attributes") or {}
        fm = self._config.field_mapping

        def get_field(name: str | None) -> str | None:
            return str(props[name]) if name and name in props else None

        def get_float(name: str | None) -> float | None:
            v = get_field(name)
            try:
                return float(v) if v is not None else None
            except (ValueError, TypeError):
                return None

        def get_date(name: str | None) -> date | None:
            v = get_field(name)
            if v is None:
                return None
            try:
                return datetime.strptime(v, self._config.date_format).date()
            except ValueError:
                return None

        return ParcelRecord(
            apn=props[fm.apn],
            geometry_geojson=feature["geometry"],
            address_normalized=get_field(fm.address),
            zoning_code_raw=get_field(fm.zoning_code),
            owner_name=get_field(fm.owner_name),
            assessed_land_value=get_float(fm.assessed_land),
            assessed_improvement_value=get_float(fm.assessed_improvement),
            last_sale_price=get_float(fm.last_sale_price),
            last_sale_date=get_date(fm.last_sale_date),
            existing_structures_count=int(props.get(fm.structures_count, 0) or 0)
                if fm.structures_count else 0,
            raw_source_data=props,
        )
```

## Task 5 — Parcel normalizer (`app/adapters/normalizer.py`)

```python
from pyproj import Geod
from shapely.geometry import shape as to_shape
from geoalchemy2.shape import from_shape

_GEOD = Geod(ellps="WGS84")
_SQM_TO_SQFT = 10.7639

def compute_area_sqft(geometry_geojson: dict) -> float:
    """Geodetic area in square feet — accurate for any EPSG:4326 polygon."""
    polygon = to_shape(geometry_geojson)
    area_m2, _ = _GEOD.geometry_area_perimeter(polygon)
    return abs(area_m2) * _SQM_TO_SQFT

def compute_centroid_wkb(geometry_geojson: dict):
    """GeoAlchemy2 WKB element for the polygon centroid."""
    return from_shape(to_shape(geometry_geojson).centroid, srid=4326)

def normalize(
    record: ParcelRecord,
    jurisdiction_id: UUID,
    zoning_district_id: UUID | None,
) -> dict:
    """Returns a dict of Parcel column values ready for SQLAlchemy upsert."""
    area_sqft = compute_area_sqft(record.geometry_geojson)
    return {
        "jurisdiction_id": jurisdiction_id,
        "apn": record.apn,
        "address_normalized": record.address_normalized,
        "geometry": from_shape(to_shape(record.geometry_geojson), srid=4326),
        "centroid": compute_centroid_wkb(record.geometry_geojson),
        "area_sqft": area_sqft,
        "area_acres": area_sqft / 43_560.0,
        "zoning_district_id": zoning_district_id,
        "zoning_code_raw": record.zoning_code_raw,
        "existing_structures_count": record.existing_structures_count,
        "assessed_land_value": record.assessed_land_value,
        "assessed_improvement_value": record.assessed_improvement_value,
        "last_sale_price": record.last_sale_price,
        "last_sale_date": record.last_sale_date,
        "owner_name": record.owner_name,
        "raw_assessor_data": record.raw_source_data,
        "data_fetched_at": datetime.now(UTC),
    }
```

## Task 6 — Zoning code mapper (`app/adapters/zoning_mapper.py`)

Reads the mapping from the `Jurisdiction` DB record (via `gis_zoning_code_map`), not from any Python file.

```python
def resolve_zoning_district_id(
    zoning_code_raw: str | None,
    jurisdiction: Jurisdiction,
    session: Session,
) -> UUID | None:
    """
    Map raw GIS zoning string → ZoningDistrict.id for this jurisdiction.
    Returns None if: raw code is None, no mapping entry, or no DB row found.
    """
    if not zoning_code_raw:
        return None
    code_map: dict = jurisdiction.gis_zoning_code_map or {}
    canonical_code = code_map.get(zoning_code_raw)
    if not canonical_code:
        return None
    row = session.execute(
        select(ZoningDistrict.id).where(
            ZoningDistrict.jurisdiction_id == jurisdiction.id,
            ZoningDistrict.code == canonical_code,
        )
    ).scalar_one_or_none()
    return row
```

## Task 7 — Ingestion service (`app/adapters/ingestion.py`)

Orchestrates the full pipeline for one parcel. Reads config from the `Jurisdiction` DB record.

```python
class ParcelIngestionService:
    def ingest_by_apn(
        self,
        apn: str,
        jurisdiction: Jurisdiction,
        session: Session,
    ) -> Parcel | None:
        """
        Fetch from county GIS, normalize, and upsert into the parcels table.
        Returns the Parcel ORM object, or None if the county has no record for this APN.
        Raises ValueError if the Jurisdiction is not configured for GIS queries.
        """
        config = JurisdictionConfig.from_orm(jurisdiction)
        adapter = ArcGISParcelAdapter(config)

        record = adapter.fetch_by_apn(apn)
        if record is None:
            return None

        zoning_district_id = resolve_zoning_district_id(
            record.zoning_code_raw, jurisdiction, session
        )
        fields = normalize(record, jurisdiction.id, zoning_district_id)

        # Upsert: update if APN already exists, insert otherwise
        existing = session.execute(
            select(Parcel).where(
                Parcel.apn == apn,
                Parcel.jurisdiction_id == jurisdiction.id,
            )
        ).scalar_one_or_none()

        if existing:
            for key, value in fields.items():
                setattr(existing, key, value)
            return existing
        else:
            parcel = Parcel(**fields)
            session.add(parcel)
            return parcel
```
